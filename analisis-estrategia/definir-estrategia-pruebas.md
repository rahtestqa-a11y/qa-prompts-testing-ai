# Definir Estrategia de Pruebas

## Objetivo

Generar una estrategia de pruebas inicial para una funcionalidad o proyecto.

## Cuándo usarlo

Úsalo cuando necesites definir alcance, tipos de prueba, riesgos, prioridades y enfoque de ejecución.
